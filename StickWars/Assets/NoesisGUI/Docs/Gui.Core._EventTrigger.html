EventTrigger

Represents a trigger that applies a set of actions (animation storyboards) in response to an event.

http://msdn.microsoft.com/en-us/library/system.windows.eventtrigger.aspx

BaseTrigger

Properties

From EventTrigger


 Name Description
Actions Gets the collection of BeginStoryboard objects that this EventTrigger maintains
RoutedEvent Gets or sets the name of the event that initiates the trigger
SourceName Gets or sets the name of the object with the event that activates this trigger. This is only used by element triggers or template triggers.

From BaseTrigger


 Name Description
EnterActions Gets a collection of TriggerAction objects to apply when the trigger object becomes active. This property does not apply to the EventTrigger class
ExitActions Gets a collection of TriggerAction objects to apply when the trigger object becomes inactive. This property does not apply to the EventTrigger class
