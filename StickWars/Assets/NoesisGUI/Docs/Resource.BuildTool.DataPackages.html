Data Packages

Data is organized in Noesis in a way very similar to Code Packages using Data Packages. As described in the SDK Structure the data directory is located at the root of the NoesisSDK distribution.

Data packages are classified by module and each module represent a directory inside the Packages directory. Inside the module directory there is a directory for each data package. For example, the data package SampleGame/Level1 belongs to the SampleGame module.

The directory Content inside each data package is where data file must be stored. This directory can be freely structured, even with subdirectories. The DB directory located at the same level than Content contains medata associated to the package. This directory is internal and must never be touched.
