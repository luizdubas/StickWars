Binding

Connects the properties of source objects (DependencyObject or implementing the INotifyPropertyChange interface) to target objects (DependencyObject). Example:

<TextBlock Text="{Binding Source={StaticResource myDataSource}, Path=PersonName}"/>

http://msdn.microsoft.com/en-us/library/system.windows.data.binding.aspx
